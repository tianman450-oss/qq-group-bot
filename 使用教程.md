# QQ 机器人使用教程（小白版）

这份教程按顺序做就行，不需要你会编程。

## 1. 这个机器人能做什么

- 群里 `@它` 聊天
- 支持识图（分析你发的图片）
- 群里发 `/画 ...` 或 `/draw ...` 生图
- 支持切换文生图模型和 LoRA
- 提示词由用户自己写，不再用固定大段提示词
- 支持导入大学课表并生成周课表图
- 上课时触发任何命令都会先有自然劝学提醒

## 2. 先准备好这 4 样

1. Windows 电脑
2. Python（建议 3.10+）
3. ComfyUI（并且能本地出图）
4. NapCat 或 Lagrange（OneBot V11）

## 3. 第一次安装（只做一次）

在项目目录打开终端，执行：

```powershell
cd <你的项目目录>
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## 4. 配置 `.env`

编辑项目根目录下的 `.env`

```env

LLM_PROVIDER="gemini"

COMMAND_ROLEPLAY_PREFIX="主、主人…千夏来给你汇报结果啦喵："

COMFYUI_WORKFLOW="workflows/text2img_default.json"
COMFYUI_IMG2IMG_WORKFLOW="workflows/img2img_no_upscale.json"
```

## 5. 启动顺序（每次开机都这样）

1. 先启动 ComfyUI（确认地址是 `127.0.0.1:8188`）
2. 再启动机器人：

```powershell
cd <你的项目目录>
.\.venv\Scripts\activate
python bot.py
```

3. 最后在 NapCat/Lagrange 配反向 WS：

- `ws://127.0.0.1:8080/onebot/v11/ws`

## 6. 命令怎么用（最重要）

### 聊天

- `@机器人 你是谁`
- 或前缀触发：`!你是谁`、`/你是谁`
- 查看命令列表：`/帮助`、`/help`、`/list`

### 识图

- 带图识别：
  - `/识图 这张图在讲什么` + 一张图片
- 回复图片识别：
  - 回复一张图发送 `/识图 这张图有什么细节`
- 也可用别名：
  - `/看图`
- 自动识图（你新要的）：
  - 在群里 `@机器人` 发消息，只要消息里带图片，机器人会自动识图
  - 回复某张图片并 `@机器人` 说话，也会自动识图

### 生图

- 默认竖图（`1024x1536`）：
  - `/画 一个雨夜街景，电影感`
- 横图（`1536x1024`）：
  - `/画 一个雨夜街景，电影感 横图`
  - 或 `/draw a rainy cyberpunk street --landscape`
- 竖图也可显式写：
  - `/画 一个雨夜街景，电影感 竖图`

### 模型 / LoRA 切换（文生图）

- 查看模型列表：
  - `/模型 列表`
- 切换模型：
  - `/模型 chenkinNoobXLCKXL_v02.safetensors`
- 查看 LoRA 列表和状态：
  - `/lora 列表`
- 切换 LoRA：
  - `/lora momoko-000052.safetensors`
- 关闭 / 开启 LoRA：
  - `/lora 关闭`
  - `/lora 开启`
- 说明：
  - 模型/LoRA切换按“每个用户”单独生效。
  - 你切换后会保留一段时间，默认 `1800` 秒（30分钟），到期自动恢复默认。
  - 可在 `.env` 里用 `DRAW_USER_STYLE_TTL_SECONDS` 调整。

### 图生图

- 图生图（消息里带图）：
  - `/图生图 赛博朋克风` + 一张图片
- 支持“回复图片触发”：
  - 回复一张图发送 `/图生图 赛博朋克风`
- 可自定义降噪 denoise（必须 `>0` 且 `<1`）：
  - `/图生图 赛博朋克风 --denoise 0.7`
  - `/图生图 赛博朋克风 denoise=0.55`
  - `/图生图 赛博朋克风 降噪0.65`
  - 不传时使用图生图工作流里的默认 denoise
- 图生图输出分辨率：
  - 默认竖图 `1536x2048`
  - 横图 `2048x1536`（在提示词里带“横图”）
- 输入图片尺寸限制：
  - 只要在 `1536x2048` 范围内都可用（短边 <= 1536，长边 <= 2048）

### 联网搜索与总结

- 主动搜索：
  - `/搜索 今天 AI 新闻`
- 群聊总结：
  - `/总结`
  - `@机器人 今天聊了啥`

### 课表解析与上课提醒

- 导入课表（推荐 WakeUp 口令）：
  - `/导入课表 493f666c20f54f4f9dd0f19ad614f228`
  - 也支持带文字的口令句子（机器人会自动提取 32 位口令）
- 兜底导入（旧方式仍可用）：
  - `/导入课表 周一 08:00-09:40 高数 @A101`
  - `/导入课表 https://你的教务课表链接`
  - 发课表截图后发送 `/导入课表`
  - 回复课表图片或 PDF 后发送 `/导入课表`
- 查看课表：
  - `/课表`
- 清空课表：
  - `/清空课表`
- 查看当前上课状态：
  - `/上课状态`
- 查看群友当前上课情况（长图）：
  - `/群友上什么课`
  - 可带数量：`/群友上什么课 30`
- 状态识别说明：
  - 机器人会按“开学日期 + 当前周次 + 单双周 + 节次”判断你是否在上课
- 全局被动提醒：
  - 当你处于上课时间，机器人执行所有命令时都会先自然劝学，再返回命令结果

### 定时推送订阅

- 订阅早报：`/订阅早报`
- 退订早报：`/退订早报`
- 订阅摸鱼图：`/订阅摸鱼图`
- 退订摸鱼图：`/退订摸鱼图`
- 手动测试：
  - `/测试早报`
  - `/测试摸鱼图`
- 早报默认来源：`Google News RSS`（可通过 `NEWS_SOURCE` 改）
- 早报可带封面图：默认使用 Bing 每日图（`NEWS_WITH_COVER=true`）

## 7. 常见问题

### Q1: 报错 10048（端口占用）

```powershell
netstat -ano | findstr :8080
taskkill /PID <占用PID> /F
```

或者把 `.env` 里 `PORT` 改成 `8081`，并把 NapCat 的 WS 同步改成 `8081`。

### Q2: 机器人不回消息

按这个顺序检查：

1. `python bot.py` 是否还在运行
2. NapCat/Lagrange 是否显示 WS 已连接
3. 是否在白名单群（如果你配置了 `GROUP_WHITELIST`）

### Q3: 生图总是失败

1. ComfyUI 是否能手动正常出图
2. `COMFYUI_SERVER` 地址是否正确
3. 工作流文件 `workflows/text2img_default.json` 是否存在

## 8. 你现在这个版本的生图规则

- 正向提示词：用户输入
- 负向提示词：沿用工作流里原来的负向词
- 模型：支持命令切换（`/模型 ...`）
- LoRA：支持命令切换与开关（`/lora ...`）
- 尺寸固定：
  - 竖图：`1024x1536`
  - 横图：`1536x1024`
- 不再支持 `--ar` 自定义比例
- 进阶能力：
  - 联网搜索（含自动搜索补充）
  - 群聊消息总结
  - 图生图
  - SQLite 长期记忆
  - 定时早报 / 摸鱼图推送（可订阅）
  - 课表导入 / 周课表渲染 / 全局上课提醒

## 9. 你必须提前准备的素材（不然功能无法完整可用）

1. 文生图工作流：`COMFYUI_WORKFLOW`
2. 图生图工作流：`COMFYUI_IMG2IMG_WORKFLOW`（必须含 `LoadImage`）

如果缺第 2 项，机器人会直接提示你缺失，不会等你测试后才“悄悄失败”。
